# A01 - California Housing Boxplot

## Project Overview
This project uses the California Housing dataset from scikit-learn to create a boxplot visualization using Python.

## Dataset
The dataset was loaded using `fetch_california_housing()` from scikit-learn.  
It contains housing information from California districts, including income, house age, population, and house value.
